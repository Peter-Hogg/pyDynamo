Manual
------

*Requires Python 3.6+*

Dependencies are the following packages:
 * attrs
 * matplotlib
 * numpy
 * pandas
 * pyqt (version 5)
 * scikit-image
 * scipy
 * tifffile (from conda-forge, if using conda. will require a conda update --all after)

Once the repository is obtained locally (either checked out, or files downloaded), navigate to the root directory and run: ::

 $> python dynamo.py

Updating
^^^^^^^^

If cloned locally using git, simply pull the new code: ::

 $> git pull

Otherwise, download the .zip file from github and replace the entire folder.
